File types and formats

Attaching files as supporting documents

To attach files:

  1. Click "Select a file". The system will open a window where you can choose a file from your computer.
  2. Provide a short description of the file (max. 100 characters).
  3. Click "Attach this file" to attach it to your request.
  4. As you attach files, they will appear in the "Selected files" window. To delete a file, click "Delete" next to the file you wish to remove.

Accepted file sizes and formats

The maximum size of each file is 2 MB (two megabytes). The total file size must not exceed 5 Mb.

The following file formats are allowed:
Portable Document Format (PDF)
JPG, TIFF, or PNG (Image)
DOC, DOCX (Microsoft Word Document)

How to scan your documents

If you are required to provide multiple pages of your passport, you must scan and save all pages as one file. Scanned documents must be clear and legible. Submitting unclear documents will result in processing delays.

Tips for reducing file size

PDF:
Recreate the PDF with Adobe Distiller set to "lowest file size."
Ensure that all images in the PDF are set to 96 Dots Per Inch (DPI).
Remove all unnecessary images.

JPG:
Scan your document at a lower DPI.
Remove all empty space around the image.
Shrink the image.
Reduce the JPG image quality.

TIFF and PNG:
Scan your document at a lower DPI.
Remove all empty space around the image.
Shrink the image.
Save the file as a JPG.

DOC, DOCX:
Remove unnecessary images, formatting and macros. Save the file as a recent Word version.
Reduce the file size of the images before they are inserted in the document.
Create a PDF document.
